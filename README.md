# 📊 Sales Data Analysis Project

## 🔍 Overview
This project performs a comprehensive sales data analysis using Python. The objective is to extract meaningful insights from raw sales data and visualize business performance across time, products, and regions.

---

## 📁 Dataset
- Source: Excel file  
- Contains information such as:
  - Date
  - Revenue / Sales
  - Products
  - Regions / Locations

---

## ⚙️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- OpenPyXL  

---

## 🧹 Data Preprocessing
- Cleaned column names  
- Converted date column to datetime format  
- Handled missing values  
- Set date as index for time-based analysis  

---

## 📈 Analysis Performed

### 1. Basic Sales Analysis
- Total Revenue  
- Total Orders  
- Average Order Value (AOV)  

### 2. Time-Based Analysis
- Daily, Monthly, Yearly trends  
- Peak sales day identification  

### 3. Product Analysis
- Top-selling products  
- Least-selling products  
- Product contribution to total sales  

### 4. Regional Analysis
- High-performing vs low-performing areas  
- Sales distribution by region  

---

## 📊 Visualizations
- Line charts (sales trends over time)  
- Bar charts (product & regional performance)  
- Comparative plots for insights  

---

## 🎯 Key Insights
- Identified peak sales periods  
- Found top-performing products  
- Highlighted regions contributing most to revenue  
- Compared high vs low performing areas  

---

## 📌 Conclusion
This project demonstrates how raw sales data can be transformed into actionable insights to support business decision-making.
